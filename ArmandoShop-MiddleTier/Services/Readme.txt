Para instalar esta aplicación junto con todo lo generado en Bin/Debug, se debe colocar a esa
altura el fichero que esta en la carpeta Spring, que contiene la definición de spring de 
gran parte de los objetos del sistema.

Ademas se debe copiar el fichero DataAccess.config, del proyecto de acceso a datos
hacia alli, este contiene la definición de la connectón string, asi como el Sql Provider usado
y el sql.
